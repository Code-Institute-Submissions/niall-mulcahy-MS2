Shape Memory Game

Working Memory Facts

  • Working memory refers to temporary workspace where we process and deal with new information

  • Reserach suggests that we can only remember 4 to 5 pieces of new information at one time

  • This can be overcome by a technique known as chunking, in which smaller pieces of information are placed into larger 'chunks' of information

  • The current Guiness World Record for Longest Sequence of Objects Memorised in One Minute is currently 54 and was set on the 19th of December, 2020.

Game Rules

  • Begin the game by pressing the start button

  • You will be shown three shapes one after another, pay attention to the order in which they come

  • Click on the shapes on the bottom of the screen in the order you were shown the shapes

  • Press on submit, if you were right the next time you play the game it will be harder!

  • If you are wrong, you go back to level 1!

  • Don't forget to have fun!
